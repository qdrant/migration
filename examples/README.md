# Examples

Ready-to-use migration script examples for common use cases.

| Example | Description |
|---|---|
| [`qdrant-to-qdrant-cloud/`](qdrant-to-qdrant-cloud/) | Migrate a self-hosted Qdrant collection into [Qdrant Cloud](https://cloud.qdrant.io) |
| [`qdrant-to-qdrant-local/`](qdrant-to-qdrant-local/) | Migrate into a self-hosted Qdrant instance (from another self-hosted instance or from Qdrant Cloud) |
